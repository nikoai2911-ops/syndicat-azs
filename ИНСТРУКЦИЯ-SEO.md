## 🔧 Что нужно сделать ВАМ (без этого поиск не заработает)

### 1. Привязать домен sd-kt.ru к сайту (DNS)

Зайдите в панель управления доменом sd-kt.ru (у регистратора — reg.ru, timeweb и т.п.), раздел **DNS-записи**.

**⚠️ ВАЖНО: НЕ удаляйте записи типа `MX` — это ваша почта @sd-kt.ru, она сломается!**

Добавьте записи для сайта:

**A-записи** (для sd-kt.ru) — четыре штуки, все на github:
```
A   @   185.199.108.153
A   @   185.199.109.153
A   @   185.199.110.153
A   @   185.199.111.153
```

**CNAME-запись** (для www):
```
CNAME   www   nikoai2911-ops.github.io
```

DNS обновляется от 1 часа до суток.

### 2. Включить домен в настройках GitHub Pages

1. Откройте репозиторий → **Settings** → **Pages**
2. В поле **Custom domain** впишите `sd-kt.ru`, нажмите **Save**
3. Дождитесь зелёной галочки, поставьте флажок **Enforce HTTPS**

> Когда DNS пропишете и будете готовы переключаться — скажите мне, я верну файл `CNAME` в репозиторий (сейчас он убран, чтобы текущий адрес github.io продолжал работать).

### 3. Яндекс.Вебмастер (для Яндекса — главный в России)

1. Зайдите на **webmaster.yandex.ru** под своим Яндекс-аккаунтом
2. **Добавить сайт** → введите `https://sd-kt.ru`
3. **Подтвердите права** — проще всего способом «HTML-файл»: скачайте файл, пришлите мне — я добавлю его в сайт и запушу; затем нажмёте «Проверить»
4. После подтверждения: раздел **Индексирование → Файлы Sitemap** → добавьте `https://sd-kt.ru/sitemap.xml`
5. **Переобход страниц** — можно вручную отправить главные страницы

### 4. Google Search Console (для Google)

1. Зайдите на **search.google.com/search-console**
2. **Добавить ресурс** → «Ресурс с префиксом URL» → `https://sd-kt.ru`
3. **Подтвердите права** способом «HTML-файл» (так же пришлите мне файл) или через DNS
4. Меню **Файлы Sitemap** → добавьте `sitemap.xml`

### 5. Яндекс.Бизнес (карточка компании — для карт и локального поиска)

1. **business.yandex.ru** → создайте организацию
2. Название SYNDICAT, адрес (Екатеринбург, ул. Окружная, 88), телефон, сайт, часы работы, фото
3. Подтвердите (звонок/СМС/почта) — появитесь на Яндекс.Картах и в локальной выдаче
