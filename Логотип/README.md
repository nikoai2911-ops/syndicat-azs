# Логотип SYNDICAT

Обновлённый логотип в векторе (SVG), адаптированный под палитру нового сайта.
Эмблема (лев + корона + S) переведена из растра в чистый вектор — чёткая на любом размере.

## Цвета

| Назначение      | HEX       |
|-----------------|-----------|
| Фон (тёмный)    | `#100904` |
| Кремовый (знак) | `#ffedd7` |
| Оранжевый акцент| `#dc5000` |
| Жёлтый фирменный| `#f2e85f` |

## Варианты цвета

- **twotone** — кремовое тело + оранжевая корона. Основной вариант для нового сайта.
- **cream** — монохром кремовый. Универсальный, строгий.
- **yellow** — фирменный жёлтый.

## Файлы

- `logo-horizontal-*.svg` — знак + надпись в строку. Для шапки сайта, писем, подписей.
- `logo-vertical-*.svg` — знак сверху, надпись снизу. Для соцсетей, плакатов, документов.
- `logo-mark-*.svg` — только эмблема (иконка). Для аватарок, печати, ватермарков.
- `favicon.svg` / `favicon-cream.svg` — иконка вкладки браузера (эмблема на тёмном, скруглённый квадрат).
- `png/` — растровые экспорты с прозрачным фоном (на случай, где SVG не принимают).
- `исходники/` — оригинальные растровые версии логотипа.

## Где уже подключено

- Шапка сайта (`index.html`, `catalog.html`) — встроена эмблема twotone + надпись.
- Фавикон и apple-touch-icon на всех страницах (`index.html`, `catalog.html`, `thanks.html`).

## Как менять размер

Все файлы — SVG, масштабируются без потери качества. В шапке высота знака задаётся
в CSS: `.brand .mark{height:32px}`. Для больших баннеров используйте `logo-vertical-*`.
